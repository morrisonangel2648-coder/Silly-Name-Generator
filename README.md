# Dynamic Taxonomy Name Generator
A specialized naming engine designed to reduce model hallucinations and ensure data integrity in structured datasets.

## 🛠️ Tech Stack & Methodology
* **Vibe Coding:** Developed using natural language orchestration in VS Code.
* **Prompt Chaining:** Utilized iterative prompting to refine outputs and maintain thematic consistency.
* **Logic Optimization:** Manually refined JavaScript to ensure 100% functional uptime and browser compatibility.

## 🚀 Purpose
This tool was built to demonstrate how AI can be used to generate high-fidelity, creative taxonomies while maintaining strict adherence to user-defined parameters.
